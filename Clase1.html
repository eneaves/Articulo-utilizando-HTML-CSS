Todo lo que necesitas saber para iniciar en Javascript

En el vertiginoso mundo de la tecnología contemporánea, Javascript emerge como un pilar fundamental. Este lenguaje de programación, nacido en los albores de la web, ha evolucionado de ser simplemente un complemento para páginas estáticas a convertirse en el lenguaje de facto para el desarrollo web moderno. Su importancia radica en su versatilidad, ya que no solo es utilizado en el lado del cliente para la interactividad dinámica en los navegadores, sino que también se ha expandido al lado del servidor con plataformas como Node.js. Además, su sintaxis sencilla y su amplia gama de bibliotecas y frameworks lo hacen accesible y poderoso para una variedad de aplicaciones, desde simples sitios web hasta complejas aplicaciones empresariales y de inteligencia artificial. En un mundo cada vez más conectado y digitalizado, Javascript se posiciona como una herramienta esencial para impulsar la innovación y el progreso tecnológico.

Variables

Las variables son contenedores para almacenar valores de datos. En Javascript, las variables se declaran utilizando la palabra clave var, let o const, seguida de un nombre descriptivo y opcionalmente un valor inicial. Los nombres de las variables pueden contener letras, números, guiones bajos y signos de dólar, pero no pueden comenzar con un número. Es importante tener en cuenta que Javascript es un lenguaje de tipado dinámico, lo que significa que no es necesario especificar el tipo de datos de una variable al declararla, ya que el tipo se determina automáticamente en tiempo de ejecución. Por lo tanto, una variable puede contener un número entero, un número decimal, una cadena de texto, un valor booleano, un objeto, una función, un array, o incluso un valor nulo o indefinido.

Inicialización de variables:

La inicialización de una variable ocurre cuando se le asigna un valor por primera vez. En JavaScript, puedes inicializar una variable usando las palabras clave var, let, o const, seguidas del nombre de la variable y el valor que deseas asignar.


var nombre = "Juan"; // Inicialización de una variable llamada nombre con el valor "Juan"
let edad = 25; // Inicialización de una variable llamada edad con el valor 25
const PI = 3.14; // Inicialización de una constante llamada PI con el valor 3.14

Asignación de variables:

La asignación de variables ocurre cuando se le da un nuevo valor a una variable que ya ha sido inicializada. En JavaScript, puedes asignar un nuevo valor a una variable simplemente utilizando el operador de asignación (=).


nombre = "María"; // Asignación de un nuevo valor "María" a la variable nombre
edad = 30; // Asignación de un nuevo valor 30 a la variable edad

Es importante recordar que en el caso de las constantes (const), una vez inicializadas con un valor, no pueden ser reasignadas. Intentar reasignar una constante generará un error.

JavaScript es un lenguaje de tipado dinámico, lo que significa que no es necesario especificar el tipo de dato al inicializar una variable. El tipo de dato se infiere automáticamente del valor que se le asigna. Además, las variables pueden cambiar de tipo en cualquier momento durante la ejecución del programa.


var x = 5; // x es inicializada como un número
x = "Hola"; // x es reasignada como una cadena de texto
x = true; // x es reasignada como un booleano

En resumen, la inicialización y asignación de variables son acciones comunes en JavaScript y son esenciales para trabajar con datos en tus programas.

Tipos de datos

En JavaScript, los tipos de datos se dividen en dos categorías principales: primitivos y no primitivos (también conocidos como tipos de referencia).

Tipos de datos primitivos:

  • Number: Representa tanto enteros como números de punto flotante.
  • String: Secuencia de caracteres encerrados entre comillas simples o dobles.
  • Boolean: Representa un valor verdadero o falso.
  • Null: Representa la ausencia intencional de cualquier valor o un objeto no válido.
  • Undefined: Representa una variable que no ha sido asignada a ningún valor.
  • Symbol: Representa un identificador único e inmutable.
  • BigInt: Permite representar números enteros de tamaño arbitrario.

Tipos de datos no primitivos (tipos de referencia):

  • Object: Es una colección de pares clave-valor.
  • Array: Estructura de datos que contiene una lista ordenada de elementos.
  • Function: Los objetos de función son instancias del constructor Function.
  • Date: Representa una fecha y hora.
  • RegExp: Representa una expresión regular para buscar patrones en cadenas de texto.

JavaScript es un lenguaje de tipado dinámico, lo que significa que no es necesario declarar explícitamente el tipo de dato de una variable antes de usarla.

Alcance de variables en JavaScript

El alcance de las variables en JavaScript se refiere a las reglas que determinan dónde y cuándo una variable puede ser utilizada dentro del código. El alcance se puede dividir en dos tipos principales: alcance global y alcance local.

Alcance global:

  • Las variables declaradas fuera de cualquier función tienen un alcance global.
  • Las variables globales pueden ser accedidas desde cualquier lugar dentro del código, ya sea dentro de funciones, bloques de código o incluso otros archivos si están en el mismo ámbito global.
  • Es importante tener en cuenta que declarar demasiadas variables globales puede llevar a problemas de colisión de nombres y a un código menos mantenible.

Alcance local:

  • Las variables declaradas dentro de una función tienen un alcance local.
  • Estas variables solo son accesibles dentro de la función en la que fueron declaradas. No pueden ser accedidas desde fuera de la función.
  • Además, las variables declaradas dentro de bloques de código (por ejemplo, dentro de un `if`, `for`, `while`, etc.) también tienen alcance local y solo son accesibles dentro de ese bloque.
  • JavaScript no tiene alcance de bloque antes de ES6 (ECMAScript 2015), lo que significa que las variables declaradas dentro de un bloque con `var` tienen un alcance de función.

Por ejemplo:


    // Alcance global
    var globalVariable = 'Soy global';
    
    function exampleFunction() {
        // Alcance local
        var localVariable = 'Soy local';
        console.log(localVariable); // Imprime: 'Soy local'
        console.log(globalVariable); // Imprime: 'Soy global'
    }
    
    console.log(globalVariable); // Imprime: 'Soy global'
    console.log(localVariable); // Error: localVariable no está definida

Variables y funciones

Las funciones son bloques de código reutilizable que pueden ser llamados y ejecutados en cualquier momento dentro de un programa. En JavaScript, las funciones son ciudadanos de primera clase, lo que significa que pueden ser tratadas como cualquier otra variable.

Declaración de funciones:


function saludar(nombre) {
    console.log("Hola, " + nombre + "!");
}

Llamada a funciones:


saludar("Juan");

Funciones anónimas:


var sumar = function(a, b) {
    return a + b;
};

Funciones de flecha (Arrow functions):


var multiplicar = (a, b) => {
    return a * b;
};

Funciones como valores:


var funcionSumar = sumar;
var resultado = funcionSumar(3, 4); // resultado es igual a 7

Ámbito de las funciones:

Las variables declaradas dentro de una función están limitadas a su ámbito local y no pueden ser accedidas desde fuera de la función. Sin embargo, las funciones pueden acceder a variables definidas en ámbitos superiores a través del mecanismo de cierre (closure).

Las funciones son una parte fundamental de JavaScript y te permiten escribir código más modular, reutilizable y mantenible. Puedes usar funciones para encapsular lógica, reducir la repetición de código y mejorar la organización de tu programa.

Bucles en JavaScript

Bucle for

El bucle for se utiliza para repetir un bloque de código un número determinado de veces.

            
    for (let i = 0; i < 5; i++) {
        console.log("El valor de i es: " + i);
    }

Bucle while

El bucle while se utiliza para repetir un bloque de código mientras se cumpla una condición específica.

            
    let j = 0;
    while (j < 3) {
        console.log("El valor de j es: " + j);
        j++;
    }

Bucle do...while

El bucle do...while es similar al while, pero garantiza que el bloque de código se ejecute al menos una vez.

            
    let k = 0;
    do {
        console.log("El valor de k es: " + k);
        k++;
    } while (k < 3);

Bucle for...in

El bucle for...in se utiliza para iterar sobre las propiedades de un objeto.

            
    let persona = {
        nombre: "Juan",
        edad: 30,
        profesion: "Ingeniero"
    };
    
    for (let prop in persona) {
        console.log(prop + ": " + persona[prop]);
    }

Bucle for...of

El bucle for...of se utiliza para iterar sobre los valores de un objeto iterable (como un array).

            
    let numeros = [1, 2, 3, 4, 5];
    for (let num of numeros) {
        console.log(num);
    }

Ciclos y Condicionales

Condicionales:

Los condicionales son estructuras de control que te permiten tomar decisiones en función de una condición. En JavaScript, los condicionales se escriben utilizando la instrucción if, seguida de una expresión que se evalúa como verdadera o falsa.


if (condición) {
    // Bloque de código a ejecutar si la condición es verdadera
} else {
    // Bloque de código a ejecutar si la condición es falsa
}

También puedes encadenar múltiples condiciones usando la instrucción else if:


if (condición1) {
    // Bloque de código a ejecutar si la condición1 es verdadera
} else if (condición2) {
    // Bloque de código a ejecutar si la condición2 es verdadera
} else {
    // Bloque de código a ejecutar si ninguna de las condiciones anteriores es verdadera
}

Ciclos:

Los ciclos te permiten ejecutar un bloque de código varias veces. En JavaScript, los ciclos más comunes son el for y el while.

El ciclo for se utiliza cuando conoces de antemano la cantidad de veces que deseas que se ejecute el bloque de código:


for (inicialización; condición; incremento/decremento) {
    // Bloque de código a ejecutar en cada iteración
}

El ciclo while se utiliza cuando no conoces la cantidad exacta de iteraciones necesarias y solo deseas que el bloque de código se ejecute mientras se cumpla una condición:


while (condición) {
    // Bloque de código a ejecutar mientras la condición sea verdadera
}

Además, existe el ciclo do...while, que es similar a while pero garantiza que el bloque de código se ejecute al menos una vez antes de verificar la condición:


do {
    // Bloque de código a ejecutar al menos una vez
} while (condición);

Los ciclos son herramientas poderosas que te permiten automatizar tareas repetitivas y manipular conjuntos de datos de manera eficiente en tus programas.

Objetos en JavaScript

Un objeto en JavaScript es una colección de propiedades y métodos relacionados. Las propiedades son variables que almacenan valores, mientras que los métodos son funciones que realizan acciones específicas. Los objetos se definen utilizando llaves {}.

  const persona = { nombre: "Juan", edad: 30, saludar: function() { console.log("Hola, soy " + this.nombre); } };

Para acceder a las propiedades de un objeto, se utiliza la notación de punto o la notación de corchetes:

  console.log(persona.nombre); // "Juan" console.log(persona["edad"]); // 30 persona.saludar(); // "Hola, soy Juan"

Arreglos en JavaScript

Un arreglo en JavaScript es una colección ordenada de valores. Los arreglos se definen utilizando corchetes [].

  const frutas = ["manzana", "banana", "naranja"];

Para acceder a los elementos de un arreglo, se utiliza la notación de corchetes con el índice correspondiente (los índices comienzan en 0):

  console.log(frutas[0]); // "manzana" console.log(frutas[1]); // "banana" console.log(frutas[2]); // "naranja"

Los arreglos tienen una propiedad length que indica la cantidad de elementos que contienen:

  console.log(frutas.length); // 3

Además, los arreglos tienen varios métodos útiles, como push(), pop(), shift(), unshift(), splice(), entre otros, que permiten manipular los elementos del arreglo.
